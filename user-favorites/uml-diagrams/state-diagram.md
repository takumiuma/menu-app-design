# 状態遷移図 - お気に入りボタン

```mermaid
stateDiagram-v2
    [*] --> 未お気に入り
    未お気に入り --> 追加処理中 : ボタンクリック
    追加処理中 --> お気に入り済み : 成功
    追加処理中 --> 未お気に入り : エラー
    お気に入り済み --> 削除処理中 : ボタンクリック
    削除処理中 --> 未お気に入り : 成功
    削除処理中 --> お気に入り済み : エラー
```
